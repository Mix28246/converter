# Конвертер единиц измерения
Программа представляет собой консольный конвертер, поддерживающий 9 различных типов величин: от температуры до систем хранения данных.
## 🛠️ Технологический стек
* **Язык**: C.
* **Компилятор**: GCC.
* **Инструменты**: Git, GitHub (использование веток, PR, Code Review).
